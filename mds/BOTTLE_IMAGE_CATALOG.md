# Bottle Image Catalog (Research Assets)

Purpose:
- Provide a ready-to-use image set for wine cards so users can visually recognize bottles.
- Keep traceability: each local asset has a source URL.

Important usage note:
- These images were downloaded from official brand pages or trusted Moldova wine catalog pages for educational/demo use in this student project.
- Before production/commercial use, verify image licensing and permission with rights holders.
